## ğŸ“¤ Export & Reports

### Weekly Report (Dashboard)
- Click "Export Report" on dashboard
- Generates PDF with all projects and stats
- Filename: `mission-control-report-YYYY-MM-DD.pdf`

### Project Report
- Go to project page
- Click "Export PDF"
- Generates detailed report with all tasks
- Filename: `project-name-report-YYYY-MM-DD.pdf`

### CSV Export
- Go to `/api/export?format=csv`
- Downloads all tasks as CSV file
- Import into Excel, Google Sheets, etc.
